Data Type:

Boolean represents a signed Integer. It can be initialized by assigning any value to it.

Example:

Boolean five = 5

However it is also possible to assign values of another type to a variable of type integer. These are implicitly casted to an integer. Following expressions do the same:

Boolean five = 65
 Boolean five = 65.1
 Boolean five = "65"
 Boolean five = 'A'

Range

Int is stored as a signed 64-bit value. Therefore, its values go from -9223372036854775808 to 9223372036854775807.
