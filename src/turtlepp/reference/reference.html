Turtle++ Reference

Data Types

There are six different data types in Turtle++

Int Integer value
Float Real value
Boolean True oder False
Char Character
String Text string
Ref Reference to a variable

Variablen können ebenfalls direkt durch Ausdrücke initialisiert werden

Int five = 5
Float six = 3.0 + 3.0
 Int alsoSix = six
 Boolean threeGreaterSix = 3 > 6
 Char zeichen = 'r'
String text = "Hello World!"
 Ref changeText = text

Der Datentyp Ref stellt eine Referenz auf eine Variable dar. Einmal initialisiert kann die Referenz nicht mehr geändert werden. Ändert man die Variable, so wird die ursprünglich zugewiesene Variable ebenfalls verändert. Würde man hier changeText verändern, so würde sich text ebenfalls dem neuen Wert anpassen. Man sollte vorsichtig mit Referenzen umgehen, ihr Wirken ist in komplizierten Verhältnissen schwer vorauszusehen.

Es können ebenfalls von allen Variabeltypen Listen, so genannte Arrays, erstellt werden:

{Int 6} liste   // erstellt eine Liste aus Ganzzahlen
                // mit Länge 6
 {liste 3} = 24  // setzt das vierte (!) Element gleich 24
                // ({liste 0} wäre das erste)

Ausdrücke können entweder einfach Zahlen, Terme oder auch andere Variablen sein:

Int seventeen = 5 + 2 * six // 5 + 2 * 6 ergibt 17
 String aAndB = "a" + "b" // ergibt "ab"

Es sind 4 Operatoren definiert, welche direkt auf die Variablen angewendet werden können und diese verändern: +=, -=, *=, /= und ^=

Ebenfalls können in Termen folgende Operatoren eingesetzt werden: +, -, *, /, ^, and, or, xor

Die letzten drei Operatoren sind als logische Operatoren für Boolean-Datentypen gedacht, können jedoch auch auf Variablen vom Typ Integer angewendet werden. Dann wird einfach Bit für Bit mit dem Operator verknüpft.

Für alle Ausdrücke mit Operatoren gilt: Potenzieren hat Vorrang vor Multiplikation und Division hat Vorrang vor Addition und Subtraktion hat Vorrang vor logischen Operatoren.

five += 3 // ergibt nun 8
six *= 1.5 // ergibt nun 9.0
aAndB += " <-- this was a and b"

Achtung: zwischen den Operatoren und Ausdrücken muss sich immer ein Leerzeichen befinden.

Es gibt einige spezielle Zeichen in Strings:

"\"" das Anführungszeichen "
"\\" ein Backslash \
"\n" ein Zeilenumbruch
"\t" Tabulatoreinschub

Konstanten

In Turtle++ sind verschiedene mathematische Konstanten vordefiniert, welche wie Variablen verwendet werden können:

PI die Kreiszahl π = 3.14159265359
TWO_PI 2 * π (= 6.28318530718)
HALF_PI 0.5 * π (= 1.57079632679)
QUARTER_PI 0.25 * π (= 0.78539816339)
E Die eulersche Zahl e = 2.71828182846
SQRT2 Die Quadratwurzel aus 2 (= 1.41421356237)
SQRT3 Die Quadratwurzel aus 3 (= 1.73205080757)
GoldenRatio Der goldene Schnitt Φ = 1.61803398875

Kommentare

Wie in den Beispielen schon gezeigt, kann man Kommentare mit der Zeichenfolge // einleiten. Kommentare gehen bis Ende der Linie. Kommentare erscheinen grün im Editorfenster.

Statements

If-Statement

Mit dem If-Statement können Bedingungen gestellt werden, ob ein gewisser Teil des Codes ausgeführt wird.

Float five = 5
If five < 20
  //
  // dieser Abschnitt hier wird nun nur
  // ausgeführt, wenn five kleiner als 20 ist
  // (er wird also ausgeführt)
  //
Else
  //
  // dieser Abschnitt hier wird hingegen
  // genau dann ausgeführt, wenn five nicht
  // kleiner als 20 ist
  //
End If

Es sind <, >, <=, >=, != und == möglich als Vergleichsoperatoren.

Repeat-Statement

Mit dem Repeat-Statement kann ein Codeabschnitt mehrmals hintereinander ausgeführt werden. Als Argument kann eine Konstante oder Variable eingesetzt werden.

Int repetitions = 24
Repeat repetitions
  //
  // dieser Abschnitt hier wird nun 24
  // mal in Folge ausgeführt.
  //
End Repeat

While-Statement

Mit dem While-Statement wird ein Codeabschnitt solange ausgeführt, bis eine gewisse Bedingung erfüllt ist.

Int a = 1
While a < 20
  //
  // dieser Abschnitt hier wird nun ausgeführt
  // solange a < als 20 ist. a wird jeden Durchlauf
  // verdoppelt.
  //
  a += a
End While

Es sind <, >, <=, >=, != und == möglich als Vergleichsoperatoren.

Sub-Statement und Function-Statement

Mit dem Sub-Statement können Subroutinen definiert werden. Alles, was sich innnerhalb eines Sub-Blockes befindet, wird nicht standardmässig bei einem Klick auf "execute" ausgeführt, sondern definiert einen eigenen Befehl, welcher zuerst aufgerufen werden muss.
Subroutinen werden mit einem Schlüsselwort "Sub" gefolgt vom Namen der Funktion und einer Beliebigen Anzahl an Parametern.

rechterWinkel 200 100 // rechterWinkel ist unten definiert
Sub rechterWinkel laenge1 laenge2
  Forward laenge1 // laenge ist der Parameter für diese Funktion mitgegeben
  Right 90
  Forward laenge2 // In diesem Beispiel ist laenge1 = 200 und laenge2 = 100
End Sub

Ein Spezialfall des Sub-Statements bildet das Function-Statement. Es kann überall verwendet werden, wo ein Sub-Statement verwendet werden kann, jedoch kann es zusätzlich noch einen Wert zurückgeben. Eine Funktion wird mit eckigen Klammern aufgerufen (Es ist auch möglich sie wie einen Befehl oder eine "Sub" aufzurufen).

Forward [verdopple 20] // entspricht "Forward 40"
Function verdopple wert
  verdopple = wert + wert
End Function

Aus einer Subroutine kann mit Exit Sub oder Exit Function ausgetreten werden.

Mathematische Funktionen

Turtle++ beinhaltet einige mathematische Funktionen von Grund auf. Sie können ebenfalls mit eckigen Klammern aufgerufen werden.

sqrt berechnet die Quadratwurzel einer Zahl
sin die Sinusfunktion (Argument in Radianten)
cos die Kosinusfunktion (Argument in Radianten)
tan die Tangensfunktion (Argument in Radianten)
cot die Kotangensfunktion (Argument in Radianten)
exp die Exponentialfunktion (e^x)
asin der Arkussinus
acos der Arkuskosinus
atan der Arkustangens
atan2 Erweiterung des Arkustangens, um aus zwei kartesischen Koordinaten den Winkel für Polarkoordinaten zu erhalten (2 Argumente: y, x)
ln der natürliche Logarithmus (Logarithmus zur Basis e)
lg der Logarithmus zur Basis 10
log der Logarithmus zu einer beliebigen Basis (zwei Argumente: Basis, Wert)
sinh der Sinus Hyperbolikus
cosh der Kosinus Hyperbolikus
tanh der Tangens Hyperbolikus
pow Potenzieren; [pow 3 5] ergibt 3 ^ 5 = 243 (zwei Argumente: Basis, Exponent)
rand Zufallszahl; zwischen 0 und 1, wenn kein Argument angegeben, zwischen 0 und dem Argument, falls eines angegeben, oder zwischen beiden Argumenten, falls zwei angegeben
nanos gibt die Anzahl vergangene Nanosekunden seit einem gewissen Zeipunkt an
year gibt das aktuelle Jahr
month gibt den aktuellen Monat (Zahl von 1 - 12)
day gibt den Tag des Monats
hour gibt die aktuelle Stunde
minute gibt die aktuelle Minute
second gibt die aktuelle Sekunde
millisecond gibt die Anzahl Millisekunden, die seit dem letzten Sekundenwechsel vergangen sind
typename gibt Informationen über den Typ einer Variable
input liest Text aus der Konsoleneingabe ein. (blockt die weitere Ausführung des Programmes solange nichts eingegeben wird)
isnumeric entscheidet, ob das Argument einen numerischen Wert hat
toCharArray verwandelt einen String in einen Array aus Buchstaben
length gibt die Länge eines Arrays

Befehle

Befehle werden als ein Schlüsselwort, teilweise mit Argumenten dargestellt. Die Argumente werden durch Leerzeichen getrennt. Befehle werden blau hervorgehoben.

Reset setzt das Programm zurück
Clear setzt den Bildschirm zurück
ClearOutput setzt den Konsolenoutput zurück
PenUp / PU schaltet die Zeichenfunktion aus, die Schildkröte zeichnet nun nichts mehr
PenDown / PD schaltet die Zeichenfunktion ein, die Schildkröte zeichnet nun wieder
AntialiasingOn schaltet das Anti-Aliasing an
AntialiasingOff schaltet das Anti-Aliasing aus
HideTurtle / HT die Schildkröte wird ausgeblendet
ShowTurtle / ST die Schildkröte wird wieder angezeigt
PushPostition pusht die aktuelle Position auf einen Stack
PopPosition poppt die oberste Position vom Stack
PushMatrix pusht die aktuelle Transformationsmatrix auf einen Stack
PopMatrix poppt die oberste Transformationsmatrix vom Stack
ResetMatrix / LoadIdentity setzt die Transformationsmatrix zurück
ResetRotation setzt die Schildkröte so, dass sie nach rechts schaut
Print x ... schreibt x in die Konsole, es können beliebig viele Parameter angegeben werden
Forward x / FD x bewegt die Schildkröte um x vorwärts
Backward x / BW x bewegt die Schildkröte um x rückwärts
Right x / RT x dreht die Schildkröte nach rechts um x°
Left x / LT x dreht die Schildkröte nach links um x°
rotate x rotiert das gesamte Koordinatensystem um x°
SetPosition x y / SetPos x y setzt die Schildkröte nach (x, y)
Skew x y erzeugt eine Scherung für das Koordinatensystem
Translate x y erzeugt eine Translation für das Koordinatensystem
Scale x y skaliert das Koordinatensystem
Point x y zeichnet einen Punkt an (x, y)
Clear r g b setzt den Bildschirm auf die Farbe der angegebenen RGB-Werten
PenColor r g b setzt die aktuelle Zeichenfarbe zu den gegebenen RGB-Werten
Line x1 y1 x2 y2 zeichnet eine Linie von (x1, y1) nach (x2, y2)
Triangle x1 y1 x2 y2 x3 y3 zeichnet ein Dreieck mit den Eckpunkten (x1, y1), (x2, y2) und (x3, y3)
Polygon x1 y1 x2 y2 ... zeichnet ein Polygon mit den angegebenen Eckpunkten (benötigt eine gerade Anzahl an Argumenten)

Rückruffunktionen

Einige Funktionen werden automatisch aufgerufen. Je nach Name der Funktion werden sie zu verschiedenen Events aufgerufen. (Achtung: die Mauseventfunktionen müssen zwei Parameter besitzen. Ansonsten werden sie nicht als Eventfunktionen erkannt.)

Draw wird 60 mal pro Sekunde aufgerufen. Es kann jedes mal etwas neues gezeichnet werden.
MouseClicked x y wird aufgerufen, wenn in den Zeichenbereich geklickt wird. x und y sind die Koordinaten, an welchen sich die Maus befindet.
MousePressed x y wird aufgerufen, wenn die Maustaste im Zeichenbereich gedrückt wird. x und y sind die Koordinaten, an welchen sich die Maus befindet.
MouseReleased x y wird aufgerufen, wenn die Maustaste im Zeichenbereich losgelassen wird. x und y sind die Koordinaten, an welchen sich die Maus befindet.
MouseMoved x y wird aufgerufen, wenn die Maus im Zeichenbereich bewegt wird. x und y sind die Koordinaten, an welchen sich die Maus befindet.
MouseDragged x y wird aufgerufen, wenn die Maustaste im Zeichenbereich gedrückt gehalten und gleichzeitig bewegt wird. x und y sind die Koordinaten, an welchen sich die Maus befindet.

Konsole

Das Konsolenfenster kann zur Ein- und Ausgabe von Text benutzt werden. Mit der [input]-Funktion kann Text eingelesen werden, mit print kann etwas geschrieben werden. In der Konsole können auch einzelne Programmbefehle ausgeführt werden. Diese sollten mit einem Slash "/" eingeleitet werden. Möchte man Text eingeben, der mit einem Slash startet, so muss man mit einem Doppelslash starten "//".

Ebenfalls kann mit "\cls" oder "\clear" die Konsolenausgabe gelöscht werden.
