            
            <p>
			<span style="color: rgb(0, 0, 255); font-family: 'courier new',courier,monospace">
            Int</span> represents a signed Integer. It can be initialized by 
            assigning any value to it.</p>

		<p>
			Example:</p>
		<p>
			<span style="font-family:courier new,courier,monospace;"><span style="color: rgb(0, 0, 255);">Int&nbsp;</span>five<span style="color: rgb(0, 0, 255);"> </span>=<span style="color: rgb(0, 0, 255);"> <span style="color: rgb(255, 0, 0);">5</span></span></span></p>
		<p>
			However it is also possible to assign values of another type to a 
            variable of type integer. These are implicitly casted to an integer. 
            Following expressions do the same:</p>
		<p>
			<span style="font-family:courier new,courier,monospace;"><span style="color: rgb(0, 0, 255);">Int&nbsp;</span>five<span style="color: rgb(0, 0, 255);"> </span>=<span style="color: rgb(0, 0, 255);"> </span>
            <span style="color: #FF0000">65<br>
            </span><span style="color: rgb(0, 0, 255);">Int&nbsp;</span>five<span style="color: rgb(0, 0, 255);"> </span>=<span style="color: rgb(0, 0, 255);"> </span>
            <span style="color: #FF0000">65.1<br>
            </span><span style="color: rgb(0, 0, 255);">Int&nbsp;</span>five<span style="color: rgb(0, 0, 255);"> </span>=<span style="color: rgb(0, 0, 255);"> </span>
            <span style="font-family: 'courier new',courier,monospace">
            <font color="#000080">&quot;65&quot;</font></span><span style="color: rgb(0, 0, 255);"><span style="color: rgb(255, 0, 0);"><br>
            </span>Int&nbsp;</span>five<span style="color: rgb(0, 0, 255);"> </span>=<span style="color: rgb(0, 0, 255);"> </span></span>
			<span style="font-family:courier new,courier,monospace;">
            <font color="#000080">'A'</font></span></p>

	    <h4>Range</h4>
        <p>
			<span style="color: rgb(0, 0, 255); font-family: 'courier new',courier,monospace">
            Int</span> is stored as a signed 64-bit value. Therefore, its values 
        go from
        <span style="color: #FF0000; font-family: 'courier new',courier,monospace">
        -9223372036854775808</span> to
        <span style="color: #FF0000; font-family: 'courier new',courier,monospace">
        9223372036854775807</span>.</p>
